Instalação da API ICP-Bravo

A API ICP-Bravo é distribuida em arquivo formato zip.

Para a instalação da API, é necessário descompactar o zip 'ICPBravoDistrib-X.XX.zip' em seu diretório de projetos, e fazer referência os pacotes a API em seus projetos.

Dentro do zip, deverá ser encontrada a seguinte estrutura:

  • doc

    diretório contendo toda a documentação da API.
    • index.html - Página inicial para consulta à documentação da API.
    • Indice.html - Índice dos capítulos da documentação.
    • Inicio.html - Apresentação da API ICPBravo.
    • desenvolvedor.html - Manual do desenvolvedor de aplicações com a API ICPBravo.
    • exemplosApplet.html - Documentação do applet da API.
    • exemplosAppletUsuario.html - Documentação do applet com interface de usuário.
    • instalacao.html - Documentação de instalação da API.
    • integracao.html - Manual de integração com hardwares criptográficos.
    • javadoc - diretório com o conteúdo javadoc da API.
    • ICPBravo.js - conjunto de biblioteca em javascript para ser utilizado em páginas HTML/JSP/PHP.
    • seguranca.html - Descrição dos mecanismos de seguraça da API.
    • caracteristicasTecnicas.html - Descrição das características técnicas da API.
    • suporte.html - Relação de Telefones/Endereços para atendimento e suporte técnico.


  • lib

    diretório contendo os arquivos necessários para compilar programas com a API ICPBravo.

    • Para programas em JAVA

    • ICPBravoAPI-X.XX.jar - JAR com a API ICPBravo.
    • commons-httpclient.jar, commons-codec.jar e commons-logging.jar - API para chamadas HTTP. Necessário para acessar o CRL.
    • mail.jar - API da Sun que possibilita o envio de emails. Necessário para o uso de S/MIME.


    • Para programas em outras linguagens com acesso via DLL

    • MsCAPI.dll - DLL para acesso ao conjunto de bibliotecas da Microsoft, denominada CapiCOM.
    • Oak.ICPBravo.dll (*) - DLL ICPBravo para desenvolvimento de aplicação servidor em ambiente .NET.
    • (*) - Este arquivo está presente apenas na distribuição para versão .NET

  • sample

    diretório contendo o código fonte de exemplos da API.
    • br/com/oaks/ICPBravo/Exemplos.java - Conjunto de códigos fontes que demonstram a utilização da API.


  • img

    diretório contendo imagens e figuras utilizada na API e na documentação da API.


  • pages, scripts, styles, WEB-INF

    diretório contendo exemplos de uma aplicação em JSP, possibilitando o entendimento de aplicações em ambiente WEB utilizando a API.

Para verificar os sistemas operacionais e versões de navegadores e JREs suportados, vide Características Técnicas



A instalação dos applets nas páginas e/ou da API em projetos será abordada nos capítulos pertinentes ao desenvolvimento de aplicações.
